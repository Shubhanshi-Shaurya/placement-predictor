# Placement Predictor 

A Machine Learning project that predicts the likelihood of a student getting placed based on parameters like CGPA and IQ.

##  Overview
This repository contains a Google Colab notebook that performs data analysis and builds a prediction model using Scikit-Learn.

##  Dataset
The model uses a dataset containing:
* **CGPA:** Cumulative Grade Point Average of the student.
* **IQ:** Intelligence Quotient score.
* **Placement Status:** Whether the student was placed (1) or not (0).

##  Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
* **Environment:** Google Colab 

##  How to Use
1. Clone this repository.
2. Open `placement_predictor.ipynb` in Google Colab 
3. Run the cells to see the data visualization and model training process.
